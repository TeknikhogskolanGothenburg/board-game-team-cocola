# User stories

Use this template when writing the user stories:

"As a < type of user >, I want < some goal > so that < some reason >."

Create a new Markdown file for each userstory and create a toc.yml in the userstories folder, and add each userstory to the TOC (table of content).

Read more on userstories at [https://www.mountaingoatsoftware.com/agile/user-stories
](https://www.mountaingoatsoftware.com/agile/user-stories
)

# Our userstory
* Som utvecklare vill jag skapa spelet Fia för att klara projektet.
* Som utvecklare av Fia vill jag skapa en tärning, spelgubbar och spelbräda för att spelet ska kunna starta och pågå.
* Som utvecklare av Fia vill jag ha fyra stationer/ fängelser, fyra spelgubbar och en väg som gubbarna ska följa för att spelet ska följa regler och vara roligt att spela.
* Som utvecklare vill jag ha fyra färger på fängelsen och dess gubbar för att skilja dessa åt enklare.
* Som utvecklare vill jag skapa fyra vägar till mål och ett mål för att det ska vara möjligt att vinna spelet.  
* Som spelare vill jag kasta en tärning och få en sexa för att börja spela.
* Som spelare måste jag stå kvar i fängelset om jag inte får en sexa första gången.
* Som spelare med en eller flera gubbar ute ur fängelset vill jag kunna välja att ta ut en till gubbe ur fängelset när jag slår en sexa på tärningen, för att ha fler gubbar i spelet.
* Som spelare får jag komma ut ur fängelset om jag får en sexa när jag kastar tärningen och får också kasta tärning en gång till för att gå med min gubbe.
* Som spelare med flera gubbar ute ur fängelset vill jag välja vilken gubbe som ska röra sig när jag slår tärningen för att t.ex. förhindra att bli kastad tillbaka till fängelset (se punkterna nedan).
* Som gubbe får jag gå lika många positioner som siffror på tärningen för att röra min gubbe framåt.
* Som gubbe i spelet måste jag gå moturs för att följa spelets regler.
* Som gubbe i spelet kan jag slå ut eller slås ut av andra gubbar och går tillbaka till fängelset om min position i spelet krockar med en annan gubbes position när jag kastar tärningen för att göra spelet konkurranskraftigt.
* Som gubbe vill jag gå i mål när jag har gått runt hela cirkeln för att kunna vinna.
* Som gubbe i slutposition vill jag få lika många siffror på tärningen som det är positioner kvar till målet för att vinna spelet.
* Som gubbe i slutposition får jag inte röra mig om jag får fler siffror i tärningen än det är positioner kvar till målet för att följa reglerna.
* Som tärning vill jag kunna slumpa fram siffran när jag kastar tärningen för att spelet ska vara rättvist.
